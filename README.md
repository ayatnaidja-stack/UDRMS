# University Dormitory & Restaurant Management System (UDRMS)

A C++ Object-Oriented project for ENSIA that manages university dormitories, 
rooms, students, and restaurant services, with a Qt GUI.

## Features
- Student management (add, view, remove)
- Dormitory and room management
- Restaurant menu management
- File-based data persistence
- Qt-based graphical interface

## Classes
- Student
- Room
- Restaurant
- Dormitory
- University

## How to build
Open the project in Qt Creator and run
